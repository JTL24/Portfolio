Project Details

A Calcutta sports auction is a type of auction where participants bid on teams or players in a sports tournament. The highest bidder owns the team or player and can win a share of a prize pool based on the performance of their team or player in the tournament. The prize pool is typically distributed according to a predefined structure, rewarding the owners of the best-performing teams or players. This format adds a gambling element to the sports event, enhancing excitement and engagement for the participants. This project is inspired by a Full Stack Web Development using the MERN (MongoDB, Express, React, Node) tech stack project I previously created. I decided to turn it into an app to increase the outreach for this project. It involves the development and implementation of a React Native app using Node, Express, and PostgreSQL designed for Calcutta sports auctions, covering events such as March Madness, golf, and NFL/NBA playoffs. The app aims to streamline the auction process in a Calcutta betting pool, providing an intuitive and engaging user experience for friends and Calcutta enthusiasts.

The project includes the following key features:

  • User Interface Design
  • Mobile Development
  • Full Stack Development
